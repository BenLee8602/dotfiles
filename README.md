# dotfiles
my config files for apps and desktop env

## packages
- alacritty (terminal emulator)
- i3 (window manager)
- nvim (text editor)
- picom (x compositor)
- polybar (status bar)

## installation
use `dotman.sh` to manage dotfiles
```
git clone https://github.com/BenLee8602/dotfiles.git dotfiles
cd dotfiles
./dotman.sh install
```
can also uninstall
```
./dotman.sh uninstall
```
